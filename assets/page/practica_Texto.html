Más Formato de Texto

Etiqueta <q>: Se usa para citas cortas que se muestran en línea. La imaginación es más importante que el conocimiento.

Etiqueta <cite>: Se usa para citar el título de una obra. Ejemplo: La pintura La noche estrellada es de Van Gogh.

Etiqueta <dfn>: Se usa para definir un término por primera vez en un documento. Ejemplo: La HTML es el lenguaje estándar para crear páginas web.

Etiqueta <abbr>: Se usa para abreviaturas o acrónimos, con el título completo en un atributo `title`. Ejemplo: ONU.

Etiqueta <code>: Se usa para mostrar un fragmento de código. Ejemplo: console.log('Hola, Mundo');.

Etiqueta <pre>: Se usa para texto preformateado, conservando espacios y saltos de línea. Muy útil para bloques de código más largos.

function saludo() {
  return "Hola";
}

Etiqueta <small>: Se usa para texto que debe mostrarse más pequeño, como derechos de autor o notas al pie. Ejemplo: Copyright 2025.

Etiqueta <ins>S: Se usa para marcar texto que ha sido insertado en el documento. A menudo se muestra subrayado. Ejemplo: Texto nuevo añadido.
